# Lou Enterprises

Corporate website for Lou Enterprises, showcasing the portfolio of brands including Lou Casa, Lou Canette, and Lou Ji.

## Features

- Responsive navigation with animated menu
- Image gallery with brand showcase
- About and Contact pages
- Custom animations and transitions
- Optimized WebP images

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS
- React

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Structure

```
app/
  ├── page.tsx          # Homepage with intro animation
  ├── about/            # About page
  └── contact/          # Contact page
components/
  ├── Navbar.tsx        # Navigation component
  ├── Footer.tsx        # Footer component
  ├── Gallery.tsx       # Image gallery
  └── IntroAnimation.tsx # Homepage animation
public/images/          # Brand images and logos
```
